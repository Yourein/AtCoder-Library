# AtCoder-Library
基本自作（基本なので例外もあります）のAtCoderのライブラリを保管しています。  
高速化に成功したアルゴリズムや理解して次回以降は  
0から書くと時間のかかるアルゴリズムの大枠などをまとめています。

## BFS
|コード名|アルゴリズム|用途 ・ 説明|
|:--:|:--:|:--:|
|BinaryIndexedTree.py|転倒数|転倒数を求められる|

|コード名|アルゴリズム|用途 ・ 説明|
|:--:|:--:|:--:|

|コード名|アルゴリズム|用途 ・ 説明|
